Dart从入门到放弃 - 关键字

发表于 2018-11-27 | 分类于 学习

因为看银河护卫队2拖到现在才开始，居然毫无罪恶感。
今天学习Dart中的关键词，看了下官方文档总共有58个，比我想象中的少。

内置标识符

开发中我们一定要避免使用内置标识符作为标识符。如果尝试为类或类型名使用内置标识符，则会发生编译时错误。
abstract import as dynamic
interface export library external
mixin factory typedef operator
part covariant get set
static deferred implements

有限的保留字

Dart的1.0版本之后添加的与异步支持相关的更新的、有限的保留字。
sync* async async*
yield await

普通保留词

无论如何我们都不能在编码中使用保留词作为标识符。
do super in switch
assert else enum is
this throw true break
extends new try case
null catch false var
class final void const
finally rethrow while continue
for return with default
if
