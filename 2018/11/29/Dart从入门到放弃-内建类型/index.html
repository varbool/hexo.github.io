Dart从入门到放弃 - 内建类型

发表于 2018-11-29 | 分类于 学习

今天开始学习Dart的内建类型，下列是Dart的常见的内建类型:

  • numbers
  • strings
  • booleans
  • lists
  • maps
  • runes
  • symbols

在Dart中我们可以使用字面量初始化任何上述类型的对象，例如，’hello word’是字符串字面量，而true是一个布尔类型的字面量。
Dart中的每个变量都指向一个对象，通常情况下我们可以使用构造函数来初始化变量。有一些内置类型有自己的构造函数。例如，可以使用Map()构造函数创建映射。

numbers

看官方文档说Dart中的数字只有int和double两种形式；吓得我吃了一鲸，居然没有我们熟悉的float。

int

Dart中的int会根据平台的不同，最大值和最小值是不同的，在Dart VM中，值可以从-2的63次方到2的63 - 1。编译成JavaScript的Dart使用JavaScript代码，允许值从-2的53次方到2的53 - 1。

Runes (字符)

在Dart中，字符是字符串的UTF-32编码点。因为Dart字符串是UTF-16代码单元的序列，所以在字符串中表示32位的Unicode值需要特殊的语法。
表示Unicode码点的常用方法是\uXXXX，其中XXXX是4位数的十六进制值。例如,心型字符(♥)的编码为\ u2665。要指定大于或小于4位十六进制数字，请将值放在花括号中。例如笑脸表情(😆)的编码\u{1f600}。
Dart中的String类有几个属性可以用来获取runes信息。codeUnitAt和codeUnit属性返回16位代码单元。使用字符属性获取字符串的字符。
