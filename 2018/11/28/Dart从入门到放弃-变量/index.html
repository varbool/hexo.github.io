Dart从入门到放弃 - 变量

发表于 2018-11-28 | 分类于 学习

我们一定要在资本主义压榨和剥削的时间里提升自己 - 维克多·鲁迅。

为了晚上有时间练吉他和看书，一大早到公司就开始学习了。今天要学习的知识点是Dart的变量。
先举一个栗子：
var name = 'Dany';
上面代码的意思是生成一个名为name的变量,引用的是一个值为Dany的字符串对象。
String name = 'Dany';
通过Dart的类型推断，name的类型被推断为String，我们可以通过指定其他类型来更改类型。
dynamic name = 'Dany;'
如果我们的变量类型不限于单一类型；我们需要用dynamic关键词来指定对象为动态类型。

关于变量，有两个比较重要的知识点，默认值和Final和const修饰符

默认值

未初始化的变量的初始值是null，甚至具有数字类型的变量也是null，这是因为Dart中的数字和其他东西一样都是对象，这一点和一些语言不一样；很多语言数字类型的初始值都会被赋值为0。

Final 和const修饰符

在Dart中，如果我们不打算改变一个变量，我们可以使用final或const修饰他，最终变量只能被赋值一次。为了方便理解，我们直接用代码来解读。

const变量是一个编译时常数，最终的顶级或类变量在第一次使用时被初始化，对于那些我们想要在编译时确定并且不再改变的变量，使用const关键词修饰。如果const变量位于类级别，则将其标记为静态const。在声明该变量时，将该值设置为编译时常量，例如数字或字符串字面量、const变量或常量数字算数运算的结果：

const关键词不只是声明常量变量，我们还可以用它来创建常量值，以及声明创建常量值的构造函数。任何变量都可以赋值一个常量值。

我们可以从const声明的初始化表达式中省略const，比如下面的这一段代码:
const baz = []; // 等同于 "const []"
我们可以更改一个非final的非const变量的值，即使他曾经有一个const值：
foo = [1, 2, 3]; // foo原本是 const []
我们不能改变const变量的值:
baz = [42]; // Error: Constant variables can't be assigned a value.
