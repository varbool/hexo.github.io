Swift进阶 - 集合类型

发表于 2018-08-10 | 分类于 学习

在所有的编程语言中，元素的集合都是最重要的数据类型。在语言层面上对于不同类型的容器 的良好支持，是决定编程效率和幸福指数的重要因素。本篇博文我们将会学习 Swift 中内建的几种主要集合类型，并重点研究如何以符合语言习惯 的方式高效地使用它们。

数组和可变性

使用函数将行为参数化

在Swift的标准库中，有许多函数都支持将行为参数化，也就是说；一个函数接受调用者传入另一个函数，并将它座位函数执行的关键步骤,下面我们就列举部分此类函数：

  • map 和 flatMap 如何对元素进行变换
  • 􏰁filter 元素是否应该被包含在结果中
  • reduce 如何将元素合并到一个总和的值中
  • sequence 序列中下一个元素应该是什么?
  • forEach 对于一个元素，应该执行怎样的操作
  • sort，lexicographicCompare 和 partition 两个元素应该以怎样的顺序进行排列
  • index，􏰁rst和contains 元素是否符合某个条件
  • min 和 max 两个元素中的最小/最大值是哪个
  • elementsEqual 和 starts 两个元素是否相等
  • split 这个元素是否是一个分割符
  • pre􏰁x 当判断为真的时候，将元素滤出到结果中。一旦不为真，就将剩余的抛弃。和 fi􏰁lter 类似，但是会提前退出。这个函数在处理无限序列或者是延迟计算 (lazily-computed) 的序列时会非常有用。
  • drop 当判断为真的时候，丢弃元素。一旦不为真，返回将其余的元素。和 pre􏰁fix(while:) 类似，不过返回相反的集合。
