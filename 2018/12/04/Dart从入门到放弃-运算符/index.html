赋值操作符

在Dart中我们可以使用=来赋值，如果我们仅仅为非空的变量赋值可以使用??=操作符.

除了=操作符来赋值之外，Dart还提供了很多复合赋值操作符，如+=将操作和赋值合并在一起构成复合赋值操作符:
= -= /= %= >>= ^=
+= *= ~/= <<= &= |=

条件表达式

条件表达式最主要的作用就是简化代码很多需要if-else语句的表达式：
condition ? expr1 : expr2 // 如果条件为真则计算expr1，或者计算expr2
expr1 ?? expr2 // 如果expr1是非空则返回expr1，否则计算expr2

级联表示法 (..)

级联(..)允许我们在同一个对象上创建一个操作序列，除了函数调用之外，我们还可以访问同一个对象上的字段。通常情况下可以使我们省去创建临时变量的步骤，使我们更为流畅的写代码。
严格意义上来说，级联表示法不是运算符，只是Dart语法的一部分。
